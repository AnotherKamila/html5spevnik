html5spevnik by AnotherKamila

A neat, touch-friendly and human-friendly songbook (and maybe one day a general system for organizing stuff) written as an HTML5 offline app

This thing should have an almost paper-like feeling, at least in being minimal (so you will see only the things you want to see on the screen, not a huge amount of distractions like in common digital songbooks). The main reason why I am doing this is that I want to try to write a semi-serious web app, learn about all the new html5 awesomeness and finally get the impression that I know Javascript. The annotated source code is at http://anotherkamila.github.com/html5spevnik/docs/ . It is not planned to be in a working state anytime soon.
